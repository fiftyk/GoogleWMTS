# 变更历史

## 1.2.0

*2018-06-24*

### 新增

- 增加环境变量：
  - `TILE_TYPE`, 默认值  `default`, 可选值：`GOOGLE`
    当 设置 TILE_TYPE=GOOGLE 时，支持如下结构的 TILE：
    ```
    /L11/R00000328
    - C00000698.png
    - C00000699.png
    - C0000069a.png
    - C0000069b.png
    - C0000069c.png
    - C0000069d.png
    - C0000069e.png
    - C0000069e.png
    ```
     

## 1.1.0

*2018-05-21*

### 新增

- 增加环境变量：
  - `HOST`， 用于设置服务主机地址，默认值 `127.0.0.1`；
  - `POST`， 用于设置服务端口号，默认值 `5555`；
